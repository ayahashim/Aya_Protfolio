    Machine translation for Arabic dialects

    The objective of this project is to improve machine translation for unlabelled Arabic dialects (Iraqi, Levantine, Gulf, Nile Basin and North Africa) to modern standard Arabic, using unsupervised domain adaptation (UDA) and backtranslation (BT) techniques. In particular, I want to:

    • Improve the translation of the most distinctive Arabic dialect (the North Africa dialect) using an UDA method. I found that North Africa is the most distinct dialect by deploying dialect identification task, where pretrained language models (BERT-base, AraBERT-large and MARBERT) were used to cluster dialects by similarity without supervision.
    • Investigate several data selection methods (Domain Cosine, domain finetune and teacher-student) for BT;
    • Deploy BT to improve the performance of a translation model;
    • Compare RNN and RNN+UDA with tranformer and tranformer+UDA translation models;
    • Compare multi-dialect with single-dialect models;
    • Benchmark our proposed models to models from the literature;
    • Combine the UDA technique with BT and perform ablation on these components.
    Skills: Deep learning, Natural language processing, pretrained language models, Machine Translation, Transformer, RNN (GRU), unsupervised domain adaptation and Clustering.
    Tools: Python, Pandas, Numpy, Pytorch, scikit-learn, and BERT.

    Detect phishing links using machine learning

    Links are extracted from website and classified using classification models. Three machine learning models were used to detect phishing links, where different features are extracted from URL such as URL length, IP address, if URL has "@" symbl and others. LSTM is also used for classification, which is a sequence to sequence model. The results show that LSTM outperform classical machine learning models. This project is published in IEEE xplore.
    Skills: Deep learning, Natural language processing, machine learning, classification, SVM, Random Forest, Logistic Regression, and RNN (LSTM)
    Tools: Python, Pandas, Numpy, Tensorflow and scikit-learn.
